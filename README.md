# Conexión a la API de Gemini con Python 🚀

Este repositorio contiene un script automatizado en Python para conectarse a la API de Gemini utilizando el SDK oficial moderno (`google-genai`). Las credenciales se manejan de forma segura a través de variables de entorno locales.

## 🛠️ Requisitos e Instalación
